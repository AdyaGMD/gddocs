# HSV

HSV in Geometry Dash is split up into 5 components. Each component is split with an `a`

Example: `0a1a0a0.37a1`

## Structure

| Index | Type | value |
|:------|:-----|:------|
| `0` | **Integer** | Hue |
| `1` | **Float** | Saturation |
| `2` | **Float** | Value/Brightness |
| `3` | **Bool** | S_Enabled |
| `4` | **Bool** | V_Enabled |
